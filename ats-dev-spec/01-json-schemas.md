# JSON Field Schemas

Every `Json?` field in the Prisma schema has a strict shape. Validate with Zod on both backend (API layer) and frontend (form submission).

---

## RoundTemplate fields

### `guidedPrompts` (used when `submissionMode === "guided"`)

```typescript
type GuidedPrompt = {
  id: string;          // cuid, generated by designer form
  label: string;       // "What did you build?"
  placeholder: string; // "Describe the tool you created..."
  charLimit?: number;  // optional max characters, null = no limit
};

// Field value: GuidedPrompt[]
// Min 2 prompts, max 10
// Each label must be non-empty, max 200 chars
// Each placeholder must be non-empty, max 500 chars
// charLimit if present must be >= 50
```

### `selfAssessmentDimensions` (used when `reflectionTrigger === "self_assessment"`)

```typescript
type SelfAssessmentDimension = {
  id: string;           // cuid
  label: string;        // "Clarity"
  description: string;  // "How clearly did you communicate your thinking?"
};

// Field value: SelfAssessmentDimension[]
// Min 2 dimensions, max 5
// Candidate rates each dimension 1-5 (scale is fixed, not configurable)
// Labels max 100 chars, descriptions max 300 chars
```

### `evaluationDimensions` (used by recruiter during review)

```typescript
type EvaluationDimension = {
  id: string;           // cuid
  label: string;        // "Self-awareness"
  description: string;  // "Does the candidate show honest self-reflection?"
  maxScore: number;     // typically 5 or 10
};

// Field value: EvaluationDimension[]
// Min 1 dimension, max 10
// maxScore must be 5 or 10
// Labels max 100 chars, descriptions max 300 chars
```

## Submission fields

### `guidedResponses` (populated when round's `submissionMode === "guided"`)

```typescript
type GuidedResponse = {
  promptId: string;  // matches GuidedPrompt.id from the template
  label: string;     // copied from GuidedPrompt.label (denormalized for readability)
  response: string;  // candidate's answer
};

// Field value: GuidedResponse[]
// On submit: every promptId from the template must have a non-empty response
// charLimit enforcement: response.length <= prompt.charLimit (if set)
```

### `selfAssessmentRatings` (populated when round's `reflectionTrigger === "self_assessment"`)

```typescript
type SelfAssessmentRating = {
  dimensionId: string;  // matches SelfAssessmentDimension.id from the template
  label: string;        // copied from dimension label
  rating: number;       // 1-5 integer
};

// Field value: SelfAssessmentRating[]
// On submit: every dimensionId from the template must have a rating
// Rating must be integer 1-5
```

## RoundEvaluation fields

### `dimensionScores` (recruiter fills this when reviewing a submission)

```typescript
type DimensionScore = {
  dimensionId: string;  // matches EvaluationDimension.id from the template
  label: string;        // copied from dimension label
  score: number;        // 1 to maxScore
  note?: string;        // optional per-dimension note, max 500 chars
};

// Field value: DimensionScore[]
// Every dimensionId from the template must have a score
// Score must be integer, 1 to dimension.maxScore
```

---

## ProcessVariant fields

### `roundOverrides` (per-round config tweaks for A/B testing)

```typescript
type RoundOverride = {
  roundOrderIndex: number;  // which round to override (1-based)
  overrides: {
    submissionMode?: "freeform" | "guided" | "response";
    reflectionTrigger?: "none" | "gemini_self_feedback" | "self_assessment" | "post_submission_prompt";
    candidateMediaRequirement?: "none" | "optional_voice" | "required_voice" | "required_photo" | "required_video";
    interRoundContent?: string;
    interRoundMediaUrl?: string;
  };
};

// Field value: RoundOverride[]
// roundOrderIndex must exist in the parent SelectionProcess
// Only the specified override fields are changed; all others inherit from the base round
// Cannot override: orderIndex, deadlineHours, roundTemplateId (structural fields stay fixed)
```

## Zod schemas (backend + frontend)

Create at:
- Backend: `src/api/pdgms/ats/validations.js`
- Frontend: `src/validations/ats/ats.validations.ts`

```typescript
import { z } from "zod";

export const guidedPromptSchema = z.object({
  id: z.string(),
  label: z.string().min(1).max(200),
  placeholder: z.string().min(1).max(500),
  charLimit: z.number().int().min(50).optional(),
});

export const selfAssessmentDimensionSchema = z.object({
  id: z.string(),
  label: z.string().min(1).max(100),
  description: z.string().min(1).max(300),
});

export const evaluationDimensionSchema = z.object({
  id: z.string(),
  label: z.string().min(1).max(100),
  description: z.string().min(1).max(300),
  maxScore: z.union([z.literal(5), z.literal(10)]),
});

export const guidedResponseSchema = z.object({
  promptId: z.string(),
  label: z.string(),
  response: z.string().min(1),
});

export const selfAssessmentRatingSchema = z.object({
  dimensionId: z.string(),
  label: z.string(),
  rating: z.number().int().min(1).max(5),
});

export const dimensionScoreSchema = z.object({
  dimensionId: z.string(),
  label: z.string(),
  score: z.number().int().min(1),
  note: z.string().max(500).optional(),
});

export const roundOverrideSchema = z.object({
  roundOrderIndex: z.number().int().min(1),
  overrides: z.object({
    submissionMode: z.enum(["freeform", "guided", "response"]).optional(),
    reflectionTrigger: z.enum(["none", "gemini_self_feedback", "self_assessment", "post_submission_prompt"]).optional(),
    candidateMediaRequirement: z.enum(["none", "optional_voice", "required_voice", "required_photo", "required_video"]).optional(),
    interRoundContent: z.string().optional(),
    interRoundMediaUrl: z.string().url().optional(),
  }),
});
```
